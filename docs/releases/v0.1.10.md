# NovaSDR v0.1.10

## Highlights

- Configurable default SSB passband edges (`ssb_lowcut_hz`, `ssb_highcut_hz`) in `config/receivers.json`.
- Setup/configure wizard prompts for SSB low/high cut when default modulation is USB/LSB.
- Frontend applies server-provided SSB defaults on USB/LSB mode changes.
- Background image: supports `background.jpg` or `background.png` reliably.

## Config changes

In `config/receivers.json` under `receivers[].input.defaults`:

- `ssb_lowcut_hz` (default: `300`)
- `ssb_highcut_hz` (default: `3000`)

Example:

```json
{
  "defaults": {
    "frequency": -1,
    "modulation": "USB",
    "ssb_lowcut_hz": 100,
    "ssb_highcut_hz": 2800
  }
}
```
