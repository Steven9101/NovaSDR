The UI lives in `frontend/` (React + shadcn/ui) and is served from `server.html_root`.

## Receiver layout

The receiver UI is designed to be fully usable on desktop without page scrolling:

- Waterfall header (status + bands/bookmark/share)
- Waterfall with frequency scale and passband tuner bar
- Control panels grid (demodulation, audio, waterfall controls, chat, server info)

See `docs/FRONTEND.md` for protocol and decoding details.
